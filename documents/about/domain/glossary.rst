=====================
Термины и определения
=====================

.. glossary::

   Кредитная заявка
     Проект внесения изменений в лимит группы связанных компаний.

   Требование к кредитной заявке
     Требование, которое может выставить бизнес-пользователь к кредитной заявке на основании кредитной политики или сложившейся практики. Например, кредитный аналитик может потребовать предоставить какие-то документы клиента.

   Клиент банка
     Юридическое или, в более редких случаях, физическое лицо, либо группа связанных компаний.

   Компания
     Юридическое лицо, входящее в группу связанных компаний.

   Группа связанных компаний
     Условное объединение юридических лиц, обычно по признаку управления, используемое банком для упрощения работы с клиентами.

   Подгруппа
     Чем-либо полезное объединение юридических лиц и других подгрупп внутри группы связанных компаний, используемое банком для упрощения работы с клиентами.

   Структура группы
     Древовидная структура, служащая для наглядного представления организации юридических лиц внутри группы. Состоит из подгрупп и компаний.

   Лимит группы
     Максимальная величина кредитного риска, которую готов принять банк в отношении конкретной группы связанных компаний без учёта стоимости и качества обеспечения. Простыми словами, все продуктовые лимиты, связанные с участниками группы компаний.

   Структурный лимит
     Продуктовый или контейнерный лимит.

   Продуктовый лимит
     Лимит на конкретный кредитный продукт.

   Контейнерный лимит
     Объединение продуктовых и других контейнерных лимитов, далее ограничивающее их.

   Структурирование лимита группы
     Процесс формирования структуры подчинения структурных лимитов в лимите группы и определения параметров этих структурных лимитов в рамках кредитной заявки.

   Бизнес-сегмент клиента
     Условное деление клиентов банка по их суммарной годовой выручке. Для корпоративных клиентов используется суммарная выручка всей группы. micro, small, medium, large и т.п.
